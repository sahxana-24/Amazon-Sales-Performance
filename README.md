Amazon Sales Analytics Dashboard (Power BI)

Project Overview

This project presents an interactive Amazon Sales Analytics Dashboard developed in Microsoft Power BI to analyze sales performance, profitability, customer trends, and delivery insights. The dashboard transforms raw sales data into meaningful business intelligence using Power Query, DAX, data modeling, and interactive visualizations to support data-driven decision-making.

Business Objective

The objective of this project is to analyze Amazon sales data, evaluate business performance, identify top-performing brands and product categories, monitor delivery efficiency, and generate actionable insights through an interactive Power BI dashboard.

Project Workflow

• Imported the Amazon sales dataset into Power BI.
• Cleaned and transformed the data using Power Query.
• Built relationships through data modeling.
• Created DAX measures for KPI calculations.
• Designed interactive charts and KPI cards.
• Added filters for dynamic analysis.
• Generated business insights through dashboard visualizations.
