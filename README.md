 🛒 Amazon Sales Analytics Dashboard (Power BI)

A professional **Amazon Sales Analytics Dashboard** developed in **Power BI** to analyze sales performance, profitability, customer trends, and delivery insights. The dashboard leverages Power Query, DAX, and interactive visualizations to transform raw sales data into meaningful business intelligence, enabling data-driven decision-making.


 🚀 Features

- 💰 KPI Cards for Total Sales, Total Profit, and Delivery Days
- 📈 Revenue by Brand Analysis
- 🏷️ Brand-wise Sales Performance
- 🚚 Delivery Status Distribution
- 📦 Order Status Analysis
- 📊 Profit Contribution by Category
- ⭐ Sales vs Customer Rating Analysis
- 🎛️ Interactive Filters for Brand and Product

---

🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel Dataset
- Data Visualization

📂 Dataset

The dashboard is built using a structured Amazon sales dataset containing:
- Brand
- Product Name
- Category
- Sales
- Profit
- Customer Rating
- Delivery Status
- Order Status
- Delivery Days

📈 Business Insights

- Identify top-performing brands based on revenue.
- Compare sales performance across brands.
- Analyze profit contribution by product category.
- Monitor delivery and order status.
- Evaluate customer ratings alongside sales performance.
- Gain actionable insights through interactive filtering.
